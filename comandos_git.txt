git init . -> cria um repositório git na pasta atual
git status -> mostra o status do trabalho
git add nome_arquivo -> adiciona arquivo para ser commutado
git commit -m "msn" -> faz commit do que estava pra ser commmitado
git log -> mostra todo o histórico de commits
git diff -> mostra o que foi alterado desde o ultimo commit do arquivo analisado
git diff . -> mostra as alerações em todos os arquivos do repositório
git reset -> remove  um arquivo adicionado à lista de prontos para o commit
git checkout -b nome da branche -> cria uma branche secundária
git checkout -D nome da branch -> Deleta uma branch estando fora dela
git checkout nome da branch -> muda de branch
q -> sai do git log
git merge nome da branch -> junta uma branch à outra (main)
git remote add origin "endereço do repositório -> linka o meu repositório local para o repositório no gitHub
git push origin main -> clona nosso repositório local e salva no github
git --help -> mostra os comandos e o que eles fazem
git remote set-url origin (url do diretório no git) -> edita a url caso tenha inserido a url errada
git remote -v -> mostra os repositórios remotos conectados ao projeto
git config --global --list -> mostra o usuário e email conectados no git
touch nome_arquivo.extensão -> cria um arquivo limpo
git pull origin main -> clona o repositório remoto e salva no local

## Fluxo de trabalho
1. criar diretório (pasta) do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. editar ou criar arquivos
4. git status
5. git add <nome do aqruivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta ao passo 3
