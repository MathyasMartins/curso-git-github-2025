git init . -> cria um repositório git na pasta atual
git status -> mostra o status do trabalho
git add nome_arquivo -> adiciona arquivo para ser commutado
git commit -m "msn" -> faz commit do que estava pra ser commmitado
git log -> mostra todo o histórico de commits
git diff -> mostra o que foi alterado desde o ultimo commit do arquivo analisado
git diff . -> mostra as alerações em todos os arquivos do repositório
git reset -> remove  um arquivo adicionado à lista de prontos para o commit
## Editor de Textos no git
nano -> editor de textos no git
## Fluxo de trabalho
1. criar diretório (pasta) do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. editar ou criar arquivos
4. git status
5. git add <nome do aqruivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta ao passo 3
